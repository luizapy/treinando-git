# 📚 Treinando Git

Este repositório foi criado com o objetivo de praticar os principais conceitos de **Git** e **GitHub**, desenvolvendo familiaridade com versionamento de código e fluxo de trabalho utilizado no desenvolvimento de software.

## 🎯 Objetivo

Aprender, praticar e consolidar os comandos e conceitos fundamentais do Git, além de entender como utilizar o GitHub para armazenar, compartilhar e colaborar em projetos.

## 📖 Conteúdo estudado

Durante os estudos foram abordados os seguintes tópicos:

* Introdução ao Git
* O que é controle de versão
* Instalação do Git
* Configuração inicial
* Criação de repositórios
* Commits
* Histórico de versões
* Branches
* Merge
* Repositórios remotos
* Push e Pull
* Clone
* Fork
* Pull Request
* Trabalho colaborativo utilizando GitHub

## 🛠️ Tecnologias

* Git
* GitHub

## 📂 Estrutura

Este repositório contém exercícios e testes realizados durante o aprendizado, servindo como ambiente de prática para comandos e fluxos de versionamento.

## 🚀 Objetivo futuro

Este projeto faz parte da minha jornada de estudos em desenvolvimento Back-end. A intenção é dominar o Git e o GitHub para utilizar boas práticas de versionamento em projetos pessoais e profissionais.

## 📚 Curso utilizado

Os estudos foram realizados utilizando a playlist de vídeos "Curso de Git e Github Para Iniciantes", do canal **Programação Web**.
